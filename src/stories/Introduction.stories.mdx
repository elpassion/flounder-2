# El design system
El design system is a UI component library made with tailwind css based on [**TACO**](https://www.figma.com/file/xy6otn2JWHNdF70Tuq0UbS/TACO-Design-System-%5BDRAFT%5D?node-id=2054%3A3026&t=4KYtpNsJBJG5fIry-1).
It supposed to help out develop project using ready to use components and corresponding design. There should not be any component logic, just a styled markup.


# How to use
1. [**Add tailwindcss to your project**](https://tailwindcss.com/docs/installation)
2. [**Copy tailwind.config.js**](https://github.com/elpassion/design-system/blob/main/tailwind.config.js)
3. Browse components and copy source code from `docs` tab
